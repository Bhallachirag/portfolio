VaxFlow 5 microservices

View repo ↗

Vaccine booking and inventory platform split into 5 independent microservices — Auth, Vaccine & Search, Booking, Reminder, and an API Gateway — each with its own MySQL database via Sequelize, talking to each other over internal REST calls. The Gateway verifies every request against the Auth service before proxying it on, and applies rate limiting to protect downstream services.

The Booking service computes order cost with FIFO batch logic — it pulls all inventory batches for a vaccine, sorts by expiry date, and consumes the nearest-to-expire stock first, so pricing always reflects the batch actually being used. Checkout runs through a Razorpay payment link; a webhook confirms the payment, deducts inventory across every item in the cart, and triggers the Reminder service to email a confirmation. Frontend is React + Vite with Zustand stores for auth, cart, and vaccine state. Deployed on GCP.

MicroGig Marketplace

View repo ↗

MERN micro-internship marketplace where clients post gigs and freelancers apply with a bid, delivery time, and portfolio link. Job search supports category, skill, budget range, and duration filters with server-side pagination and sort-by-budget or sort-by-urgency.

Payments run escrow-style: a Razorpay order is only created once a job is marked APPROVED and READY_FOR_RELEASE, and the server independently re-verifies the payment using an HMAC-SHA256 signature check before releasing funds — so a tampered client response can't fake a payout. Reviews feed back into a running average rating recalculated on the user document after every submission. Also built an AI drafting assistant on Gemini that turns a freelancer's rough notes into a structured job title and description in one click.

Hangman Game

View repo ↗

Classic Hangman built with React Router-driven flow — a start screen where the player sets a custom word or phrase (with a show/hide toggle since the word is entered by a second player), and a play screen with a masked-word display and an on-screen A–Z keyboard.

Guessed letters are tracked in state and colour-coded live — correct guesses turn green, wrong ones turn red and get disabled — while an SVG hangman figure builds up stroke by stroke with each wrong guess, driven off the same guess count as the rest of the game logic.
